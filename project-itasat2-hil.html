ITASAT-2 Hardware-in-the-Loop Roadmap for AOCS

ITA Space Center (CEI) · E2MoC Lab · Ongoing

What it is

This project defines an end-to-end hardware-in-the-loop (HIL) implementation roadmap for the ITASAT-2 microsatellite Attitude and Orbit Control Subsystem (AOCS). The roadmap connects simulation, emulation, and verification deliverables into a cohesive flow that can be executed with COTS components and realistic embedded constraints.

Motivation

Small satellites often face limited budget, schedule pressure, and high integration risk. A structured HIL plan reduces verification uncertainty by allowing closed-loop testing earlier, validating interfaces, and improving confidence in control performance before later integration stages.

My contribution

  • Defined the implementation plan and verification flow linking simulation, emulation, and test campaigns.
  • Applied a hybrid PMBOK approach to structure work packages, WBS, and a Gantt schedule for team execution.
  • Started an attitude and orbit control simulator in MATLAB using mission-representative dynamics and data.
  • Specified STM development boards to emulate a flat-sat architecture for closed-loop testing.
  • Aligned technical outputs with verification deliverables and project constraints.

Results

  • Baseline HIL roadmap and verification plan drafted and validated with stakeholder inputs.
  • Initial MATLAB simulation baseline established to support test development and iteration.
  • Hardware emulation direction defined (STM-based flat-sat), enabling next integration steps.
  • Project remains ongoing, prioritizing simulation verisimilitude and physical emulation readiness.
